# Navielektro Current Site

Clean transfer folder containing only the website created in this chat.

## Contents

```text
navielektro-current-site/
├── index.html
├── style.css
├── script.js
├── README.md
└── assets/
    └── navielektro-logo.webp
```

## What This Is

This is a static HTML, CSS and JavaScript website. It does not use React, Next.js, Vite or npm dependencies.

The design includes:

- Homepage with hero, about section, products, services, industries, references, news and contact CTA
- Products page with real Navielektro product categories and product detail views
- Services, Industries, About, References, News and Contact pages
- Responsive navigation, search, breadcrumbs and mobile menu
- Visual placeholders for future images, videos, renders and photography

## Run Locally

You can open `index.html` directly in a browser.

For a local server preview:

```bash
python3 -m http.server 4175
```

Then open:

```text
http://127.0.0.1:4175/index.html
```

## Build

No build step is required.

To deploy, upload:

```text
index.html
style.css
script.js
assets/
```

## Notes

Most company-specific copy, product specifications, contact details and reference/customer information are intentionally marked as placeholders until approved final content is available.
